# JSONEasy

## Development

1. Clone this repo
2. Download [JSON5](https://unpkg.com/json5/dist/index.min.js) into `libs` as `json5.min.js`.
3. Load unpacked the repo's root directory in `chrome://extensions/`.

## Packaging

1. Run `zip -r jsoneasy.zip . -x "README.md" "tests/*" "docs/*" ".git/*"`
2. Upload `jsoneasy.zip` to [Chrome Web Store](https://chrome.google.com/webstore/developer/dashboard).
